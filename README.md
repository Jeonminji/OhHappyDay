# OhHappyDay
개발 기간: 2019. 07 ~ 08 <br>
개발 인원: 4명 <br>
개발 언어: Java, C++ <br>
<br><br>
<b>💡 주요 기능 </b> <br>
1) 아두이노에 연결된 센서를 이용해 현재 위치에서 온도, 습도와 미세먼지 농도 측정 <br>
2) 센서로 측정된 온도에 따른 옷 추천 <br>
3) 앱과 연결된 공기청정기를 On/Off 해서 미세먼지 농도 조절 가능
